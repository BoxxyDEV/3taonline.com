Buying a PlayStation 3 and Jailbreaking it

There are currently 2 exploits for the PS3.

PS3HEN (Homebrew ENabler)
Released: May, 2019
Pros

  • Less likely to get banned from PlayStation Network (not really an issue anymore)
  • Works on all PS3 Models (Fat, Slim, & Super Slims)

Cons

  • Most mods on PS3 aren't supported
  • Harder to sign games with licence files
  • Cannot run Linux
  • Cannot downgrade firmware

CFW (Custom FirmWare)
Released: March, 2011
Pros

  • Seeing as the Exploit is older. there is alot of mod support and much more articles to help that only work on CFW
  • Can bring back to OtherOS which Sony removed in the famous update 3.21
  • Easier to make mods for (more support and tools)

Cons

  • maybe 0.1% chance to get banned (but really easy to get around and Sony has stop seeming to ban consoles/accounts anymore since Rockstar Games shutdown official servers on December 16th, 2021)
  • Only Supports Fat Models, & only the PS3 Slim (All CECH-20xx, CECH-21xx, & some of the CECH-25xx models)

Make sure you backup any modded files if you want to save them for later

⇨ Disc version

1. Uninstall old game updates

2. Install the PKG for your version (usb or /dev_hdd0/packages)

if you are installing from USB make sure your USB is formatted in FAT32 (if it is over 32gb you can try fat32format or Rufus to format your flash drive)
